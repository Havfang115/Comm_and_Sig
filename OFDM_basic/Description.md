# OFDM 系统在多径衰落信道下的性能仿真技术报告
## 引言
正交频分复用（OFDM）通过将高速数据流分解为多个低速子载波传输，结合循环前缀（CP）有效对抗多径衰落，广泛应用于 4G/5G 等无线通信系统。本报告基于 Python 仿真，分析 QPSK 调制的 OFDM 系统在多径衰落与加性高斯白噪声（AWGN）信道下的误比特率（BER）性能。
## 系统模型
### 1. 发射机
**比特生成**：随机生成N_sub×Bps×N_symbols个二进制比特（N_sub为子载波数，Bps=2为 QPSK 每符号比特数，N_symbols为 OFDM 符号数）。
**QPSK 映射**：将比特对映射为复符号，并归一化功率（/√2）。
**OFDM 调制**：将符号分组为N_symbols×N_sub块，经 IFFT 转换为时域信号，添加长度为Cp_len的循环前缀，最后序列化输出。
### 2. 信道
**多径衰落**：模拟 3 径信道，延迟为[0, 2, 4]样点，衰减因子经功率归一化（||attenuation||=1）。
**AWGN 噪声**：根据目标 SNR（dB）计算噪声功率，添加复高斯白噪声（实部 / 虚部方差各为噪声功率的 1/2）。
### 3. 接收机
**同步与去 CP**：将接收信号重组为块，去除循环前缀。
**OFDM 解调**：经 FFT 转换回频域。
**QPSK 解调**：通过最小距离判决（实部 / 虚部符号判决）恢复比特对。
### 4. 结果与分析
**BER 随 SNR 增加显著下降**：当 SNR 从 5 dB 提升至 15 dB 时，BER 从约10^-1量级降至10^-3以下，符合 AWGN 信道下调制性能的理论趋势。
**多径对抗有效性**：循环前缀的添加有效抑制了多径引起的符号间干扰（ISI），使系统在多径环境下仍保持稳定性能。
### 5. 结论
本仿真验证了 OFDM 系统结合 QPSK 调制在多径衰落与 AWGN 信道下的可靠性。通过合理设置子载波数、循环前缀长度及多径归一化，可有效提升系统抗干扰能力。后续可扩展至更高阶调制（如 16QAM）或信道估计模块，进一步优化性能。
